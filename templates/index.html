AtCoder Type Checker

AtCoderのコンテスト成績から、同じ（内部）レート帯の中で「多く解くタイプ」であるか「早く解くタイプ」であるかを判定します。スコアの値が大きいほど、前者と判定されたことを意味します。

{{ message }}

{{ message1 }}

{{ message2 }}

{{ message3 }}

{{ message4 }}

用語について

  • 内部レート: こちらの記事における「レート（第二段階）」を指します。
  • 順位率: ある参加者がコンテストで丁度P点を得て、丁度P点を得た中でR位であり、丁度P点を得た人が合計N人だった場合の (R-1) / (N-1) の値です。
    同じ内部レート帯の中で比較したとき、この値が小さいことが多いほど、（多く解くことと比べて）早く解くことが得意なタイプであると考えられます。
  • 平均順位率: P>0 かつ N>1 である場合についての順位率を平均したものです（Rated コンテストのみを対象としています）。
  • 内部レートによる補正: 平均順位率は内部レートと負の相関があり、その相関を取り除くための補正です。「その内部レートの人が取ると考えられる平均順位率」を表します。
  • スコア: ((平均順位率) - (内部レートによる補正)) * 100 です。

補足

  • 現在は、下記の最終更新日時までに行われた、ナンバリングされているABC・ARC・AGCのみを計算の対象としています。
  • 指定したIDの内部レートが300～3000の範囲外の場合、結果の信頼度が低い可能性があり、その旨が表示されます。
  • 判定結果はあくまで目安であり、その参加者の性質を保証するものではありません。また、タイプはコンテスト参加数が増えるにつれて大きく変わることがあります。
